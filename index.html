Мои навыки

HTML
CSS

Я хорошо владею самыми главными языками в создании сайтов — CSS и HTML. Они отвечают за то, какого цвета будет текст, и будет ли он вообще.

JavaScript

JavaScript - язык для интерактивности на сайтах, например заявки, навигацию, игры и разные плагины для программистов.

Figma

Figma - это онлайн-инструмент для создания макетов. Прототип сайта рисуют, и вы можете сразу видеть, каким он будет.

GitHub

GitHub - это бесплатный сервис, на котором хранятся многие сайты. Также этот сервис позволяет создавать к ним ссылки, но все равно ограниичивает в их настройке.
