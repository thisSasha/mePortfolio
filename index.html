Мои навыки

HTML

HTML - это язык, который позволяет создавать веб-страницы. Если ты знаешь HTML, то можешь оформить страницу изображениями, текстом, ссылками и другим содержимым. HTML помогает структурировать эти элементы, чтобы страница выглядела красиво и была удобной для использования.

CSS

CSS - это язык, который используется для управления тем, как выглядит веб-страница. С его помощью можно задавать цвета фона, шрифты, размеры текста, расположение элементов на странице и т.д. То есть, если HTML определяет структуру страницы, то CSS описывает ее внешний вид и красоту.

JavaScript

JavaScript – это язык программирования, который широко используется для создания веб-страниц с интерактивными элементами, такими как анимация, обработчики событий и многое другое. JavaScript позволяет добавлять функциональность на страницу, которая не может быть достигнута только с помощью HTML и CSS.

Figma

Figma - это онлайн-инструмент для создания макетов и прототипов веб- и мобильных приложений. Он позволяет дизайнерам создавать и редактировать макеты в режиме реального времени совместно с другими членами команды, что ускоряет процесс работы и улучшает коммуникацию. Когда макет готов, его можно экспортировать в различные форматы, включая PNG, JPEG и SVG. Но также есть возможность экспортировать HTML-код для веб-вёрстки.

GitHub

GitHub - это сервис для хранения, управления и совместной работы над проектами с использованием системы контроля версий Git. На GitHub можно создавать репозитории (хранилища файлов) и делиться ими с другими людьми, делать изменения в файлы, отслеживать изменения и комментировать их. Это позволяет разработчикам работать над проектами в команде, а также легко отслеживать изменения и версии проекта.
